# Projet_PHP_L1A-293_JUVENAL

Ce projet PHP a été réalisé dans le cadre d’un examen pratique afin de démontrer l’utilisation de la programmation orientée objet (POO).
Le projet consiste à créer plusieurs classes PHP comme Compte, Personne, Etudiant, Animal, Chien et Chat afin d’illustrer différents concepts de la POO tels que l’encapsulation, l’héritage, le polymorphisme et l’abstraction.

Le projet comprend également un fichier BDD.php permettant de se connecter à une base de données MySQL et un fichier index.php qui affiche les résultats avec une interface utilisateur réalisée grâce au framework Bootstrap 5.

La base de données ecole est utilisée pour stocker les informations concernant les élèves, les enseignants et les classes.
Le projet peut être exécuté sur un serveur local comme XAMPP, WAMP ou MAMP afin de tester le fonctionnement de l’application.
